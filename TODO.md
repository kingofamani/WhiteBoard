### 技術架構
- **前端框架**: 純 JavaScript (ES6+)
- **樣式框架**: TailwindCSS
- **模組化設計**: 
  - CanvasModule (核心畫布)
  - BackgroundModule (背景管理)
  - TextToolModule (文字工具)
  - NotesModule (便條紙)
  - QRCodeModule (QR code 產生器)
  - TimerModule (計時器)
- **響應式設計**: 支援桌面和行動裝置
- **資料結構**: drawingHistory 陣列儲存所有繪圖資料
