# Web 教學白板 (MVP)

## 專案簡介

本專案旨在建立一個網頁版的互動式教學白板，提供教師或簡報者一個方便的線上工具。此為 MVP (Minimum Viable Product) 版本，專注於實現核心功能。
靈感來源：classroomscreen.com

## 🎯 專案狀態

### ✅ 已完成功能 (MVP 完成)

1.  **畫布 (Canvas)**:
    *   ✅ 基本繪圖功能：
        *   畫筆 (可選顏色、粗細)
        *   橡皮擦
    *   ✅ 清空畫布功能
    *   ✅ 畫布尺寸自動填滿整個瀏覽器視窗 (100% 寬高)
    *   ✅ RWD 響應式調整，內容保留與重繪

2.  **背景管理 (Background Management)**:
    *   ✅ 允許使用者更換畫布背景：
        *   純色背景 (白色、淺灰、自訂顏色)
        *   預設圖片背景 (格線)

3.  **文字工具 (Text Tool)**:
    *   ✅ 允許使用者在畫布上新增文字框
    *   ✅ 可以輸入文字內容
    *   ✅ 文字保留和重繪功能

4.  **便條紙 (Sticky Notes)**:
    *   ✅ 允許使用者新增便條紙到畫布上
    *   ✅ 可以在便條紙上輸入文字
    *   ✅ 便條紙可以被拖拽移動
    *   ✅ 便條紙內容保留和重繪

5.  **計時器 (Timer)**:
    *   ✅ 提供倒數計時功能
    *   ✅ 提供碼錶功能
    *   ✅ 可在畫布上顯示計時器
    *   ✅ 計時器面板 UI 控制

6.  **響應式網頁設計 (RWD)**:
    *   ✅ 畫布響應式調整
    *   ✅ 內容保留和重繪機制
    *   🔄 工具列適應性佈局 (進行中)

## 功能特色

### 🎨 繪圖功能
- **多工具支援**: 畫筆、橡皮擦、文字、便條紙
- **自訂設定**: 顏色選擇、線條粗細調整
- **觸控支援**: 支援滑鼠和觸控設備操作

### 🖼️ 背景管理
- **多種背景**: 白色、淺灰、格線、自訂顏色
- **即時切換**: 背景變更時保留所有繪圖內容

### 📝 文字與便條紙
- **文字工具**: 點擊畫布任意位置新增文字
- **便條紙**: 可新增、編輯、拖拽移動的便條紙
- **內容保留**: 所有文字和便條紙在視窗調整時保持不變

### ⏱️ 計時器功能
- **倒數計時**: 可設定分鐘和秒數的倒數計時器
- **碼錶功能**: 精確到百分秒的碼錶計時
- **畫布顯示**: 可選擇在畫布右上角顯示計時器

### 📱 響應式設計
- **自適應畫布**: 畫布自動填滿螢幕並響應視窗大小變化
- **內容保留**: 視窗調整時自動重繪所有內容
- **跨設備支援**: 支援桌機、平板、手機使用

### 📱 QR Code 產生器 (QR)
- **真實 QR Code**: 使用 qr-server.com API 生成可掃描的 QR code
- **多種內容支援**: 支援文字、網址、電話號碼等各種內容
- **自訂大小**: 可調整 QR code 大小（100-300px）
- **即時預覽**: 輸入內容後立即顯示 QR code 預覽
- **點擊放置**: 在畫布上點擊任意位置放置 QR code
- **錯誤處理**: 網路問題時自動使用備用方案
- **內容保留**: QR code 會保存在畫布歷史中，支援重繪

## 使用方法

### 基本操作
1. **選擇工具**: 點擊工具列上的按鈕選擇畫筆、橡皮擦、文字或便條紙工具
2. **調整設定**: 使用顏色選擇器和粗細滑桿調整繪圖參數
3. **開始繪圖**: 在畫布上點擊或拖拽進行繪圖

### 文字功能
1. 點擊文字工具按鈕 (T)
2. 在畫布上點擊要新增文字的位置
3. 在彈出的文字框中輸入內容
4. 按 Enter 或點擊其他地方完成輸入

### 便條紙功能
1. 點擊便條紙工具按鈕 (N)
2. 在畫布上點擊要新增便條紙的位置
3. 在便條紙中輸入文字內容
4. 拖拽便條紙可移動位置
5. 清空文字內容可刪除便條紙

### 計時器功能
1. 點擊計時器按鈕 (⏱) 開啟計時器面板
2. 選擇倒數計時或碼錶模式
3. 設定時間並開始計時
4. 可選擇在畫布上顯示計時器

### 背景管理
1. 使用背景下拉選單選擇預設背景
2. 選擇「自訂顏色」可設定任意背景顏色
3. 背景變更時會保留所有繪圖內容

## 快速開始

1. **下載專案**:
   ```bash
   git clone [repository-url]
   cd WhiteBoard
   ```

2. **啟動本地伺服器**:
   ```bash
   # 使用 Python
   python -m http.server 5000
   
   # 或使用 Node.js
   npx serve .
   ```

3. **開啟瀏覽器**:
   訪問 `http://localhost:5000`

## 🎯 專案特色

- **🎨 完整繪圖功能**: 畫筆、橡皮擦、顏色選擇、線條粗細調整
- **🖼️ 多樣背景選項**: 白色、淺灰、格線、自訂顏色背景
- **📝 文字工具**: 點擊畫布新增可編輯文字
- **📋 便條紙系統**: 可拖拽移動的便條紙，支援文字編輯
- **📱 QR Code 產生器**: 輸入文字或網址生成 QR code
- **⏱️ 計時器功能**: 倒數計時器和碼錶，支援畫布顯示
- **📱 響應式設計**: 支援桌面和行動裝置，畫布填滿螢幕 100%

## 🚀 功能說明

### 基本繪圖工具
- **畫筆 (P)**: 自由繪圖，支援顏色和粗細調整
- **橡皮擦 (E)**: 擦除繪圖內容
- **清空畫布**: 一鍵清除所有內容

### 背景管理
- **白色背景**: 純白色背景
- **淺灰背景**: 淺灰色背景
- **格線背景**: 方格線背景，適合繪製圖表
- **自訂顏色**: 選擇任意背景顏色

### 文字工具 (T)
- 點擊畫布任意位置新增文字
- 支援文字編輯和樣式調整
- 文字內容會保留在畫布歷史中

### 便條紙 (N)
- 點擊畫布新增便條紙
- 支援文字編輯和內容修改
- 可拖拽移動到任意位置
- 多種顏色選擇

### QR Code 產生器 (QR)
1. 點擊 QR code 工具按鈕 (QR)
2. 在彈出的面板中輸入文字或網址
3. 即時預覽生成的 QR code
4. 調整 QR code 大小（100-300px）
5. 點擊「生成 QR Code」按鈕
6. 在畫布上點擊要放置 QR code 的位置
7. 生成的 QR code 可用手機掃描使用

### 計時器 (⏱)
- **倒數計時器**: 設定分鐘和秒數進行倒數
- **碼錶**: 精確到百分秒的計時功能
- **畫布顯示**: 可將計時器顯示在畫布上
- **視覺提醒**: 倒數結束時閃爍提醒

## 🛠️ 技術架構

### 核心模組
- **CanvasModule**: 畫布核心功能，處理繪圖和事件
- **BackgroundModule**: 背景管理和繪製
- **TextToolModule**: 文字工具功能
- **NotesModule**: 便條紙系統
- **QRCodeModule**: QR code 產生和管理
- **TimerModule**: 計時器功能

### 技術棧
- **前端**: 純 JavaScript (ES6+)
- **樣式**: TailwindCSS (CDN)
- **畫布**: HTML5 Canvas API
- **響應式**: CSS + JavaScript RWD
- **QR Code**: qr-server.com API

### 資料結構
```javascript
drawingHistory = [
  {
    tool: 'pen|eraser|text|note|qrcode',
    // 畫筆/橡皮擦
    points: [[x1,y1], [x2,y2], ...],
    color: '#000000',
    lineWidth: 5,
    
    // 文字
    text: 'Hello World',
    x: 100, y: 100,
    font: '16px Arial',
    
    // 便條紙
    width: 200, height: 150,
    textColor: '#000000',
    fontSize: 14,
    
    // QR Code
    dataURL: 'data:image/png;base64,...',
    width: 100, height: 100
  }
]
```
